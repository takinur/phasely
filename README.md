# Phasely

Phasely is a Chrome extension that detects job application fields and autofills them from a structured profile.

## Development

```bash
npm run dev
npm run build
npm run lint
npm run test
```

## Load extension locally

1. Build the project with `npm run build`.
2. Open `chrome://extensions`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and choose the generated `dist/` folder.
